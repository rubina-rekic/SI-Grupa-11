# PostRoute Frontend

Minimalni React + TypeScript skeleton sa jasnom podjelom slojeva.

## Trenutna struktura
- `src/app`: glavni ulaz u aplikaciju
- `src/ui`: stranice, layout-i i prezentacijske komponente
- `src/application`: hook-ovi i orkestracija flow-a
- `src/infrastructure`: routing, API client placeholder, konfiguracija, storage
- `src/shared`: zajednicki tipovi i utility funkcije
- `src/styles`: globalni stilovi
- `src/vite-env.d.ts`: lokalni `ImportMetaEnv` tipovi

Frontend trenutno prikazuje samo skeleton UI i nije implementacija poslovnih funkcionalnosti.
